# Feature Specification: Manual Mode

**Feature Branch**: `002-manual-mode`
**Created**: 2026-02-17
**Status**: Draft
**Input**: TBD


**Цель**
Реализовать ручной режим отладки промптов и пайплайна. Это основная цель создания приложения

**Функции**
1. Редактирование файла профиля с сохранением в новый файл
2. Редактирование промптов
3. Каждый шаг запускается вручную
4. можно запустить генерацию с любого шага, если предыдущие шаги выполнены
5. Каждый шаг можно перезапустить
6. На каждом шаге можно просматривать:
   1. Промпт
   2. Какие переменные используются из файла профиля
   3. Какие данные подаются на вход
   4. Статистика выполнения - токены и время
   5. Результат - содержимое ответа со всеми полями
7. Для каждого шага можно установить дополнительные предобработчики, постобработчики, тесты запроса.
8. Можно определить порядок редобработчиков, постобработчиков, тестов.
9. Нужно спроектировать и реализовать оценку эффективности промпта - насколько он следует иснтрукциям и насколько адекватна генерация требованиям из файла профиля.
10. Пользователь проставляет собственную оценку эффективности промпта
11. результаты сохраняются, можно просматривать историю
